快速开始

欢迎使用 AICG 视频创作软件！按照以下步骤快速上手：

  1. 下载并安装软件
  2. 注册账号并登录
  3. 选择 AI 模型开始创作
  4. 导出您的作品

常见问题

Q1: 如何下载软件？

点击页面上方的"软件下载"按钮即可下载最新版本。

Q2: 软件支持哪些系统？

目前支持 Windows 11 系统，后续将支持更多平台。

Q3: 如何获取 API 密钥？

注册账号后，在个人中心可以管理您的 API 密钥。

Q4: 视频生成需要多长时间？

根据视频长度和复杂度，通常需要 1-5 分钟。

Q5: 如何联系技术支持？

可以加入 QQ 群 18737262 联系开发者。
