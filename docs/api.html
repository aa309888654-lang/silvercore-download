API 文档

通过 API 接入 AICG 视频与图片生成能力

主要接口

POST /v1/images/generations 文生图 - 根据文本描述生成图片 ▼

请求示例

{ "prompt": "一只可爱的猫咪在草地上玩耍", "model": "dall-e-3", "n": 1, "size": "1024x1024" }

请求参数

参数类型必填说明
promptstring是图片描述文本
modelstring否模型名称，默认 dall-e-3
ninteger否生成数量，默认 1
sizestring否图片尺寸，如 1024x1024
POST /v1/videos/generations 文生视频 - 根据文本描述生成视频 ▼

请求示例

{ "prompt": "日出时分，海浪拍打金色沙滩", "model": "sora", "duration": 10 }

请求参数

参数类型必填说明
promptstring是视频描述文本
modelstring否模型名称，默认 sora
durationinteger否视频时长(秒)，默认 10
POST /v1/videos/image-to-video 图生视频 - 上传图片生成视频 ▼

请求示例

{ "image_url": "https://example.com/image.jpg", "prompt": "图片中的风景在微风中轻轻摇曳", "model": "runway" }

请求参数

参数类型必填说明
image_urlstring是图片URL地址
promptstring否视频描述文本

认证方式

在请求头中携带 API Key 进行认证：

Authorization: Bearer YOUR_API_KEY
