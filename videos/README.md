# 视频文件目录

请将教程视频文件放在此目录下。

## 文件命名规范

| 视频文件 | 封面图片 | 说明 |
|---------|---------|------|
| `quickstart.mp4` | `quickstart-cover.jpg` | 快速入门教程 |
| `image-gen.mp4` | `image-gen-cover.jpg` | 图片生成教程 |
| `video-gen.mp4` | `video-gen-cover.jpg` | 视频生成教程 |
| `advanced.mp4` | `advanced-cover.jpg` | 高级技巧教程 |

## 支持的视频格式

- MP4 (推荐)
- WebM
- OGG

## 视频建议规格

- **分辨率**: 1920x1080 或 1280x720
- **编码**: H.264
- **帧率**: 30fps
- **大小**: 建议不超过 50MB

## 封面图片建议

- **格式**: JPG 或 PNG
- **分辨率**: 与视频分辨率一致
- **大小**: 建议不超过 500KB
