账户与订阅

Q如何注册 AICG 账号？

▼

访问 AICG 官网，点击右上角「立即开始」按钮，使用邮箱或手机号注册账号。注册成功后会自动获得免费试用额度，可以体验基础功能。

Q有哪些订阅套餐？如何选择？

▼

AICG 提供以下套餐：

  • 免费版：每月100次生成额度，适合尝鲜体验
  • 专业版：¥99/月，无限生成，专属客服支持
  • 企业版：面议，私有化部署，API调用，定制开发

根据你的使用频率和功能需求选择合适的套餐。

Q免费额度用完了怎么办？

▼

免费额度用完后，你可以：

  • 升级到付费套餐获得更多额度
  • 参与社区活动获取额外奖励
  • 邀请好友注册获得奖励次数

使用与操作

Q如何生成一张图片？

▼

生成图片非常简单：

  • 在输入框中描述你想要的图片内容
  • 选择图片比例（1:1、16:9、9:16等）
  • 选择图片风格（写实、动漫、油画等）
  • 点击「生成」按钮，等待几秒钟即可获得图片

Q视频生成需要多长时间？

▼

视频生成时间取决于多个因素：

  • 视频时长：5秒约30秒，10秒约1分钟，30秒约3分钟
  • 当前队列繁忙程度
  • 服务器负载情况

生成过程中可以关闭页面，之后在「历史记录」中查看和下载。

Q支持哪些图片和视频格式？

▼

图片格式：PNG、JPG、JPEG、WebP，支持最大 4096x4096 分辨率

视频格式：MP4、MOV，支持 720P 和 1080P 分辨率输出

Q生成的图片可以商用吗？

▼

付费用户生成的图片和视频可用于个人和商业用途。但请注意：

  • 不得生成侵犯他人版权的内容
  • 不得生成虚假新闻、误导性信息
  • 涉及人像时需获得授权许可

API 与技术

Q如何获取 API Key？

▼

登录账号后，进入「控制台」→「API」页面，点击「创建 API Key」即可生成。请妥善保管你的 API Key，不要泄露给他人。

QAPI 调用有频率限制吗？

▼

不同套餐有不同的频率限制：

  • 免费版：60次/分钟
  • 专业版：600次/分钟
  • 企业版：无限制，支持自定义

QSDK 支持哪些编程语言？

▼

目前官方提供 Python、JavaScript、Java、Go 四种语言的 SDK。你也可以直接调用 REST API 与我们的服务交互。

支付与退款

Q支持哪些支付方式？

▼

我们支持支付宝、微信支付、信用卡（Visa/MasterCard）、PayPal 等多种支付方式。企业用户可通过银行转账支付。

Q支持退款吗？

▼

付费套餐在购买后7天内，如对服务不满意，可申请全额退款。超过7天或已使用额度超过30%的情况下，不支持退款，但可申请转为其他套餐。
